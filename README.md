# Global Petrol Price & Consumption Analysis — June 2022

## 📌 Project Overview
Analyzed global oil consumption, petrol prices, and affordability across 180+ countries using Python (Pandas, Matplotlib, Seaborn).

## 🧹 Data Cleaning Steps
- Fixed misaligned row for Côte d'Ivoire
- Removed commas, percentage signs, and converted text numbers to numeric values
- Corrected column names and fixed encoding issues
- Standardized all units for consistent analysis

## 🔍 Key Insights
1. **United States** leads daily oil consumption at ~19.7 million barrels — more than double China at #2.
2. **Venezuela** has the cheapest petrol globally at **$0.08/gallon**, while **Hong Kong** is most expensive at **$11.35/gallon**.
3. There is **no strong link** between a country’s GDP and petrol price — rich nations can have cheap or expensive fuel.
4. **Nigeria** sells petrol at **$1.57/gallon** (far below global average), but low GDP means only ~1,336 gallons are affordable per person yearly.
5. Heavily subsidized nations (Venezuela, Libya, Iran) show extreme low prices, while high-tax developed nations have the highest rates.
